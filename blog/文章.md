## 使用教程
0. 首先安装git 只需搜索git下载安装即可

1. 登陆

2. Your repositories （可选择已有仓库 or 新建仓库）

    注意可以在本地新建仓库上传
    
        git init
        git add .
        git commit -am "add"
        git push origin master

    或者直接拷贝已有仓库，直接在网页上选择 Clone or download 然后使用 git 指令
    
        git clone（copy SSH）
